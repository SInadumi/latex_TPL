使用方法
「latexmkrc」をtexファイルが存在するディレクトリに配置する．

実行レシピ：「latexmk」
コンパイル方法:Ctrl + Alt + B
PDFの表示:Ctrl + Alt + V
コマンドパレット:F1 or Ctrl+Shift+P
コード整形:Shift+Alt+F
